Definición de HTML: es un lenguaje de marcado que se utiliza para el desarrollo de páginas de Internet. Se trata de la sigla que corresponde a HyperText Markup Language. Existen las siguientes versiones: HTML 2.0, HTML 3.2, HTML 4.01 (Strict, Transitional y Frameset), HTML 5, HTML 5.1 y HTML 5.2
